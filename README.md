# Algorithm Examples

Python implementations of classic graph algorithms, companion code for [Algorithm Pathfinder](https://algorithm-pathfinder-production.up.railway.app).

## Algorithms

| File | Algorithm | Description |
|------|-----------|-------------|
| [`dijkstra.py`](dijkstra.py) | Dijkstra's Shortest Path | Find shortest paths from source to all nodes |
| [`prim.py`](prim.py) | Prim-Jarn√≠k's MST | Build minimum spanning tree by growing from a start node |
| [`kruskal.py`](kruskal.py) | Kruskal's MST | Build minimum spanning tree by sorting and selecting edges |

## Run

```bash
python dijkstra.py
python prim.py
python kruskal.py
```
